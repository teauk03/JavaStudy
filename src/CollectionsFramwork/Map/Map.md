# **Map Collection Class**

---

* Map 인터페이스는 Collection 인터페이스와는 다른 저장 방식을 가짐
* Key-Value를 하나의 쌍으로 저장하는 방식을 사용

### **Map 컬렉션 클래스의 공통된 특징**
1. 요소의 저장 순서가 유지되지 않음
2. 키(Key)의 값은 중복을 허용하지 않지만, 값(Value)의 값은 중복을 허용함
